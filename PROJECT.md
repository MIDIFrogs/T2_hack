# T2 Schedule - Frontend Specification

## 📋 Описание проекта

Система планирования рабочего времени для сотрудников T2. Фронтенд для линейных специалистов с удобным календарным интерфейсом.

---

## 🎯 Основной сценарий

1. **Руководитель** → открывает период сбора графика (через админку, не в scope этого проекта)
2. **Сотрудник** → заполняет график на период через красивый интерфейс
3. **Руководитель** → выгружает график в Excel (через админку, не в scope)

**Важно:** Дополнительного подтверждения внесённого графика нет — это ручная работа руководителей после выгрузки.

---

## 👤 Пользователь

Сотрудник (роль `user`) с верифицированным аккаунтом.

---

## 🚀 Стек технологий

### Почему этот стек?

```
Next.js 14 + shadcn/ui + Framer Motion
```

**Преимущества:**
- ⚡ **Быстрый старт** — готовые компоненты из коробки
- 🎨 **Современный UI** — shadcn/ui построен на Radix UI (accessible)
- 🎬 **Анимации** — Framer Motion для wow-эффекта
- 📱 **Mobile-first** — адаптив из коробки
- 🔥 **Горячая замена** — быстрая разработка
- 🌐 **TypeScript** — типизация и безопасность

## 📦 Технологический стек

| Категория | Технология | Зачем |
|-----------|------------|-------|
| **Framework** | Next.js 14 (App Router) | SSR, API routes, оптимизация |

## 🚀 Quick Start

```bash
# 1. Create project
npx create-next-app@latest t2-schedule --typescript --tailwind --app

# 2. Install dependencies
cd t2-schedule
npm install framer-motion date-fns zustand @tanstack/react-query
npm install react-hook-form zod @hookform/resolvers
npm install lucide-react clsx tailwind-merge

# 3. Install shadcn/ui
npx shadcn-ui@latest init

# 4. Add components
npx shadcn-ui@latest add button
npx shadcn-ui@latest add dialog
npx shadcn-ui@latest add select
npx shadcn-ui@latest add input
npx shadcn-ui@latest add progress
npx shadcn-ui@latest add avatar
npx shadcn-ui@latest add card

# 5. Run
npm run dev
```
