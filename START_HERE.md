## 🐛 Если что-то не работает:

### Фронт не открывается:
```bash
# Останови процесс (Ctrl+C)
# Удали кеш
cd frontend
rmdir /s .next
# Запусти снова
npm run dev
```

### Бэкенд не запускается:
```bash
# Проверь что Python установлен
python --version

# Установи зависимости
pip install fastapi uvicorn sqlalchemy pydantic

# Попробуй снова
python -m uvicorn app:create_app --reload
```

### 401 Ошибка:
- Токен истёк → перелогинься
- Пользователь не существует → запусти `python setup_test_data.py`

### Нет периодов:
- Запусти `python setup_test_data.py`
- Или создай вручную через http://localhost:8000/docs
